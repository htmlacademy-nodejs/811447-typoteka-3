* Скопировать .env.example в .env

* Записать в переменные окружения хост, порт, название БД, имя пользователя и пароль

* Заполнить базу данных моками
```
npm run filldb 
```

* Установить зависимости
```
npm install
```

* Режим разработки
```
npm run start
```

* Запустить тесты
```
npm run test 
```
